## Descripción
Página web individual sobre Hip Hop en inglés con diseño neon verde.
Práctica de HTML5 + CSS3 con técnicas modernas.

## Características técnicas
- HTML5 semántico: header, main, section
- CSS Grid: 3 columnas de tarjetas que bajan a 1 en celular
- Media Queries: responsive para <992px y <600px
- Animaciones CSS: parpadeo en título + rotación de luz
- Efecto neon: text-shadow y box-shadow verde
- Normalize.css para reset de estilos

## Archivos
- hiphop.html: Estructura principal en inglés
- estilos.css: Estilos neon + animaciones + responsive  
- normalize.css: Reset CSS cross-browser
